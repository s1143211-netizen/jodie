# 貪吃蛇遊戲 (Snake Game)

## 檔案說明

- **snake.html**: 一個完整的網頁版貪吃蛇遊戲

## 功能特性

1. **基本遊戲玩法**
   - 蛇自動移動，吃到食物得分
   - 蛇碰牆或自撞遊戲結束

2. **控制方式**
   - 方向鍵 (↑↓←→) 控制移動
   - WASD 鍵控制移動
   - 空白鍵 暫停/繼續遊戲

3. **計分系統**
   - 每次吃到食物 +10 分
   - 最高分記錄在 localStorage

4. **視覺效果**
   - 漸層背景深色主題
   - 蛇身有漸層效果，蛇頭較亮
   - 食物有發光效果
   - 遊戲結束顯示最終分數

5. **遊戲結束**
   - 顯示遊戲結束畫面
   - 可選擇"再玩一次"
